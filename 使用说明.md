# ChatGPT 自动出图助手 — 使用说明书

## 它能做什么

你只需要在 PDF 里用 `【插图：描述】` 标记好需要配图的位置，插件会自动：
1. 解析 PDF，提取所有插图标记
2. 在浏览器里操控 ChatGPT（DALL-E）逐条生图
3. **（可选）** 开第二个 ChatGPT 标签页，用 GPT-4V 给每张图打分（百分制）
4. 不合格的图自动拿审核意见改 prompt 重新生成，直到达标
5. 所有图片自动下载到本地

---

## 第一步：安装插件

### Chrome
1. 地址栏输入 `chrome://extensions`，回车
2. 右上角打开 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `ChatGPT-Haibao\extension` 文件夹
5. 完成，工具栏会出现插件图标

### Edge
1. 地址栏输入 `edge://extensions`，回车
2. 左侧打开 **开发人员模式**
3. 点击 **加载解压缩的扩展**
4. 选择 `ChatGPT-Haibao\extension` 文件夹
5. 完成

---

## 第二步：准备 PDF

在你的 PDF 文档里，需要插图的地方插入标记：

```
【插图：一幅未来城市的天际线，日落时分，蓝金色调，竖版构图】
```

支持三种写法：

| 写法 | 示例 |
|------|------|
| 中文括号 + 中文冒号 | `【插图：描述内容】` |
| 中文括号 + 英文冒号 | `【插图:描述内容】` |
| 英文括号 | `[illustration: 描述内容]` |

> **注意：** 标记要放在 PDF 正文里，不要放在图片/文本框里。用 Word 或 WPS 导出 PDF 时选择「可编辑文字」模式。

---

## 第三步：使用插件

### 1. 打开 ChatGPT
在浏览器打开 `https://chatgpt.com`，确保已登录 Plus/Pro 账号。

### 2. 打开插件面板
点击工具栏的插件图标，弹出设置面板。

### 3. 创建项目（可选）
- 顶部 **项目** 下拉框可以切换项目
- 点击 **+新** 创建新项目（如「课程海报」「产品图」）
- 每个项目独立保存自己的提示词和审核设置
- PDF 文件放在 `projects/项目名/` 文件夹下管理

### 4. 填写提示词
在 **Prompt / Style requirements** 输入框写你的统一风格要求，例如：

```
专业大气风格，适合学术课程海报。
蓝金配色为主，竖版构图。
画面简洁有力，留白适当。
中英文字体清晰可读。
```

### 5. 选择 PDF → 解析
- 点击 **选择文件**，选你的 PDF
- 点击 **解析** 按钮
- 下方会显示解析结果：找到几个插图标记

### 6. 设置审核（可选）
| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用审核** | 勾选后每张图由 GPT-4V 打分 | 开启 |
| **通过线** | 分数 ≥ 此值才算通过 | 80 分 |
| **最大重试** | 不通过时最多重新生成几次 | 2 次 |

> 审核流程：生图 → GPT-4V 打分 → ≥80 分通过 → 不通过则拿修改意见重来

### 7. 开始生成
点击 **Start Auto Generate**，插件开始自动工作：
- 进度条实时显示完成情况
- 当前处理的任务会高亮
- 有审核时显示重试次数

---

## 第四步：查看结果

图片自动下载到浏览器的默认下载目录，文件夹名为 `chatgpt-posters/`。

文件命名规则：`P{页码}_{序号}_{描述}.png`

例如：`P3_5_未来城市天际线.png`

如果审核不通过重试过，文件名会带重试标记：
`P3_5_未来城市天际线_r1.png`（第 1 次重试）

## 常见问题

### Q: 解析 PDF 显示找不到标记？
- 检查标记格式是否正确：`【插图：xxx】`（注意是全角方括号）
- 确保 PDF 是文字型而非扫描型（扫描版 PDF 需要用 OCR 转换）
- 标记不要放在图片、表格或特殊文本框内

### Q: 提示「请先打开 chatgpt.com」？
- 在浏览器新开一个标签页，访问 `https://chatgpt.com` 并登录
- 插件需要至少一个已登录的 ChatGPT 标签页

### Q: 生成到一半弹出窗口关了？
- 不用担心！后台会继续执行，重新打开插件面板即可看到进度
- 如果浏览器完全重启，队列会自动恢复

### Q: 审核一直不通过怎么办？
- 调低通过线（比如改到 70 分）
- 增加最大重试次数
- 检查你的提示词是否清晰明确
- 在审核设置里自定义审核标准，让审核更宽松

### Q: 会不会被 ChatGPT 检测为机器人？
- 插件已经做了反检测处理：
  - 逐字键入，每字间隔随机 30-180ms
  - 偶尔打错字然后回删（~3% 概率）
  - 鼠标不点按钮正中心，随机偏移
  - 所有等待时间都加随机抖动
  - 模拟人思考停顿和页面滚动
- 但无法保证 100% 不被检测，请自行评估风险

### Q: 支持哪些 ChatGPT 版本？
- DALL-E 生图需要 ChatGPT Plus（$20/月）或 Pro
- AI 审核需要 GPT-4V 访问权限（Plus 自带）
- 免费版不支持，因为没有 DALL-E 和 GPT-4V

### Q: 能在 Edge 上用吗？
- 可以，Edge 是 Chromium 内核，完全兼容
- 加载方式：`edge://extensions` → 开发人员模式 → 加载解压缩的扩展

## 小技巧

1. **提示词写详细点**：DALL-E 靠文字理解，越具体效果越好。颜色、构图、光线、风格都写上。
2. **审核标准可以自定义**：在审核设置框里写你自己的标准，比如「重点关注人物手指是否正常」「文字必须清晰无乱码」。
3. **先拿一两张测试**：首次使用建议先做一个只有 1-2 个标记的测试 PDF，跑通流程再上大批量。
4. **审核关了更快**：如果你对生图质量要求不高，关掉审核可以快很多。
